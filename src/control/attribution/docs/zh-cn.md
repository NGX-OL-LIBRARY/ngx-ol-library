## 定义
> 显示与地图中的图层来源相关的所有属性的控件。这个控件是地图中包含的默认控件之一。默认情况下，它将显示在地图的右下部分，但这可以通过使用```.ol-attribution```的 css 选择器来改变。

## 何时使用
需要显示与地图中的图层源关联的所有属性。

```
import { NolAttributionControlModule } from 'ngx-ol-library/control/attribution';
```
