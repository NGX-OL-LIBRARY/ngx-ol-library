## Definition
> Utilities of coordinate, pipes and services are provided. It's based on [Coordinate](https://openlayers.org/en/latest/apidoc/module-ol_coordinate.html).

## When To Use

When coordinates need to be offset, rotated, formatted, etc.


## Import Module

```
import { NolCoordinateModule } from 'ngx-ol-library/coordinate';
```
