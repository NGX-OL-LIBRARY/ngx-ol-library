# NGX-OL-LIBRARY

A high-performance, feature-packed Angular map component library based on OpenLayers for all your mapping needs.

## 📦 Installation

We recommend using ```@angular/cli``` to install. It not only makes development easier, but also allows you to take advantage of the rich ecosystem of angular packages and tooling.


```
$ ng new PROJECT_NAME
$ cd PROJECT-NAME
$ ng add ngx-ol-library
```

> More information about ```@angular/cli``` [here](https://github.com/angular/angular-cli).

You can also install ```ngx-ol-library``` with npm or yarn:

```
$ npm install ngx-ol-library
  or
$ yarn add ngx-ol-library
```
